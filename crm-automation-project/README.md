# CRM-автоматизация

**Роль:** Системный аналитик  
**Период:** 2025 
**Результат:** Оптимизация процессов продаж и клиентского сервиса

---

## 📋 Описание проекта

Автоматизация CRM-системы для оптимизации работы с клиентами и повышения эффективности продаж.

---

## 🔧 Моя роль и задачи

- Сбор и анализ требований от бизнеса
- Разработка BPMN-диаграмм процессов
- Проектирование архитектуры системы
- Описание структуры данных и интеграций
- Подготовка презентации и дашборда

---

## 🛠️ Технологии

- **Анализ:** BPMN, UML, Use Cases
- **Документация:** Confluence, Jira
- **Данные:** SQL
