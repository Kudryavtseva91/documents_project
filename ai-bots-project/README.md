# AI-боты для автоматизации поддержки

**Роль:** Системный аналитик / Продуктовый менеджер  
**Период:** 2024-2025  
**Результат:** Снижение нагрузки на L1 на **60%**

---

## 🤖 Разработанные боты

### 1. AI-бот поддержки (RAG-based)
- **Задача:** Отвечает на вопросы пользователей на основе базы знаний
- **Технологии:** RAG, LLM, Embeddings, Confluence API
- **Результат:** Время ответа сокращено с 15 мин до 2 сек

### 2. AI-бот L1 (классификатор)
- **Задача:** Классифицирует заявки, назначает категорию и приоритет
- **Технологии:** NLP, ML-классификация, GLPI API
- **Результат:** Автоматизация 70% входящих заявок

### 3. AI-бот аналитики
- **Задача:** Строит отчёты и дашборды по запросу
- **Технологии:** SQL, Redash API, NLP
- **Результат:** Ускорение подготовки отчётов с часов до минут

### 4. AI-бот транскрибатор
- **Задача:** Расшифровка аудио/видео обращений
- **Технологии:** Whisper API, Speech-to-Text
- **Результат:** Структурирование голосовых обращений
