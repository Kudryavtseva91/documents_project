## 📊 Дашборды для мониторинга

### 1. Дашборд руководителя
- Количество заявок в разрезе статусов
- Соблюдение SLA по отделам
- Нагрузка на агентов
- Среднее время обработки

### 2. Дашборд агента
- Мои заявки (по статусам)
- Просроченные заявки
- SLA по моим заявкам
- Оценки пользователей

### 3. Дашборд отдела
- Заявки отдела
- Типичные проблемы
- Тренды по времени

---

## 🔄 Еженедельный отчёт

### Структура отчёта:
1. **Общие показатели**
   - Всего заявок за неделю
   - Закрыто заявок
   - Среднее время обработки

2. **SLA**
   - % соблюдения SLA по каждой группе
   - Нарушения SLA (причины)

3. **Нагрузка**
   - Заявок на агента
   - Распределение по типам

4. **Качество**
   - Средняя оценка
   - Количество возвратов
